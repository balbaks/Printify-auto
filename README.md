# AutoDrip 👕🤖

AI-powered Print on Demand automation store.
Built with Printify API + Shopify + Python.

## What this does:
- Auto generates designs using AI
- Auto uploads to Printify
- Auto publishes to Shopify store

## Stack:
- Python
- Printify API
- Shopify
- Render (hosting)
